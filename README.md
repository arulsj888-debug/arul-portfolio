# Arul's Portfolio

## Project Description
This portfolio is a modern website built with Next.js, showcasing projects and skills in a visually engaging manner. It features beautiful animations utilizing GSAP and Framer Motion, parallax effects, and demonstrates expertise in Python, PostgreSQL, and Hugging Face models.

## Features
- Modern and responsive design
- Smooth animations using GSAP and Framer Motion
- Parallax scrolling effects
- Skills showcase (Python, PostgreSQL, Hugging Face)
- Project highlights
- Contact form

## Tech Stack
- **Frontend:** Next.js, React, GSAP, Framer Motion
- **Backend:** Node.js
- **Database:** PostgreSQL
- **Others:** Python, Hugging Face models

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/arulsj888-debug/arul-portfolio.git
   cd arul-portfolio
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## How to Run
To run the development server, execute:
```bash
npm run dev
```
Go to `http://localhost:3000` in your browser to view the portfolio.
